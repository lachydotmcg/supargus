# Supargus Product Flow

This is the user-facing flow the desktop app should optimize for.

1. The user opens the app and is taken to Setup to input their information.
2. After setup, the user goes to Dashboard and presses a large central button: "Run full privacy check". While it runs, the app explains each task in plain English.
3. The result popup shows only relevant findings. The guided flow then takes the user to the right action screen. If a broker/website appears to have the user's data, the Data Brokers tab shows that site and gives clear options: send through Gmail app password, copy the email subject/body, or open the broker website form.
4. If something on the PC is found, the app explains exactly what it was, what it could mean, and how to remedy it. When a safe automated action is possible, the app should offer one clear button to protect the user.

The user should not have to understand internal statuses such as `needs_form`. They need to know where their data was found, why it matters, and what button to press next.
